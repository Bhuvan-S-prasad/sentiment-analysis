Project Overview

Advanced Sentiment Analysis System

Data Preparation

  • Custom dataset with game reviews and social media content
  • Multi-class sentiment classification (Positive, Negative, Neutral, Irrelevant)
  • Comprehensive text preprocessing pipeline including:
    • URL and mention removal
    • Hashtag cleaning
    • Number removal
    • HTML entity handling
    • Case normalization
    • Punctuation removal

Model Architecture

  • Built on HuggingFace Transformers framework
  • Fine-tuned transformer model for sentiment classification
  • Optimized for 4-class sentiment prediction
  • Maximum sequence length: 128 tokens
  • Softmax activation for probability distribution

Features

  • Real-time sentiment analysis
  • Confidence scores for all sentiment classes
  • Robust text preprocessing
  • GPU acceleration support
  • RESTful API integration
